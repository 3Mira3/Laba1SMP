# Лабораторна робота №1 — Node.js, Express, Jest, Supertest, dotenv

Цей проєкт демонструє базову роботу з платформою **Node.js**, фреймворком **Express**, а також використання тестових бібліотек **Jest** і **Supertest**. Реалізовано простий веб-сервер, який повертає повідомлення за маршрутом `/`, з можливістю змінювати текст відповіді через файл `.env`.

---

## Функціональність
- Сервер працює на порту **3000**.
- Маршрут `/` повертає повідомлення:
  - “Hello World” — за замовчуванням;
  - або значення змінної `MESSAGE` з `.env`.
- Є додатковий маршрут `/user`.
- Підключено тестування через Jest та Supertest.
- Використано `dotenv` для роботи зі змінними середовища.

---

## Як запустити проєкт локально

### 1. Клонувати репозиторій
```bash
git clone https://github.com/3Mira3/Laba1SMP.git
cd Laba1SMP
```
### 2. Встановити Node.js (якщо не встановлено)

Завантажити з офіційного сайту: https://nodejs.org

### 3. Встановити залежності
```bash
npm install
```
Запуск сервера
```bash
npm start
```

Відкрити у браузері:

http://localhost:3000

Запуск тестів
```bash
npm test
```
Зміна тексту відповіді через .env

Файл .env:
```bash
#MESSAGE=Випрями спину, а то сидиш у позі креветки)
```

Щоб змінити текст відповіді — розкоментувати або змінити значення:
```bash
MESSAGE=Привіт, це змінене повідомлення!
```

Перезапустити сервер.

Висновок

Проєкт демонструє роботу з Express, використання змінних середовища через dotenv, а також модульне тестування за допомогою Jest та Supertest. Завдання лабораторної роботи виконано повністю.
